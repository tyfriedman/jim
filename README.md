# JIM

Project scaffold for the JIM fitness social network.

## Structure

- `server/`: Node.js + Express backend API scaffold
- `client/`: React frontend scaffold
- `database/`: Oracle schema, migrations, and seed placeholders
- `docs/`: API and deployment documentation placeholders
- `scripts/`: utility script placeholders

## Environment Files

- Root: `.env.example`
- Server: `server/.env`, `server/.env.example`
- Client: `client/.env`, `client/.env.example`
